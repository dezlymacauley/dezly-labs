Data structures

Trying to pick the most efficient way to store your data.

Data representation methods that are used to organize data.
E.g. Arrays, Linked lists, trees

Arrays - data is ordered in a linear way, in consecutive memory locations.

Linked list - data is not stored in consecutive memory locations.

Tree - data is stored in a hierachical order

Algorithms 

Trying to find the most efficient way to solve a problem.

A procedure that contains steps for performing a task.

E.g. Sorting, searching, traversal

_______________________________________________________________________________

There are two factors that define the efficiency of an algorithm.

- The running time of the algorithm (lower is better)
- and the memory that it occupies (lower is better)

Other factors include network usage and disk usage.

_______________________________________________________________________________

## How to measure the running time of an algorithm

Asymptotic Analysis

Analyze the runninng time of the algorithm based on the input size.

You don't calculate the exact running time.

The running time of the algorithm depends on the size of the input.

n = Size of input
