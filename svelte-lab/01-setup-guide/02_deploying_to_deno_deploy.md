# Deploying to Deno Deploy
_______________________________________________________________________________

Sign up for an account on Deno Deploy using your GitHub account
```sh
https://deno.com/deploy
```
_______________________________________________________________________________

Click the `New app` button

_______________________________________________________________________________

Click the `Select user or organization` drop down menu

_______________________________________________________________________________

Click your GitHub account.
_______________________________________________________________________________

Click the `select repository` drop down menu

_______________________________________________________________________________

Click the `Create App` button

_______________________________________________________________________________

The default URL given to your site is:

`name-of-repo.github-username.deno.net`

_______________________________________________________________________________
