# grrs
_______________________________________________________________________________

Usage

```bash
grrs foobar test.txt
```

The program will print out all the lines in `test.txt` that contain 
the word `foobar`

_______________________________________________________________________________

The text after the program name are called command line arguments.

If they start with a `--` then they are called command line flags.
_______________________________________________________________________________

The operating system represents them as a list of strings.

They get separated with spaces.
