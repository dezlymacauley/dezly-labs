# Setting up C++ on Arch Linux
_______________________________________________________________________________

```bash
sudo pacman -S --needed base-devel
```
_______________________________________________________________________________

```bash
sudo pacman -S --needed gcc
```
_______________________________________________________________________________

```bash
sudo pacman -S --needed clang
```
_______________________________________________________________________________

#### Note: This is a package group that includes:
- `clang` A compiler for C and C++ projects
- `clangd` Language support for C and C++ projects
- `clang-format` Language support for C and C++ projects
